关于我们

我们是 BNBU 官方认证的学生组织，隶属于学校 创新中心（IC）！ 拾光文创 致力于为学生提供一个让创意真正落地的平台～

在这里，你可以：

在拾光文创门店学习制作餐饮和咖啡；

代表拾光文创与学校各社团合作举办活动，或把自己的小创意变成现实（我们的活动审核超级开放）；

在门店橱窗展示你的创意作品和策展能力；

如果你对学校官方纪念品设计（例如录取礼包里的周边）感兴趣，可以加入我们的 产品部；

如果你擅长人力资源或对外公共关系，我们也为你提供了契合的管理岗位。

我们不仅 open-minded（审核宽松），而且 实力满满！

拾光文创在校内拥有线下门店：图书馆附近的拾光文创与拾光书屋，都是我们的专属地盘！

BNBU有众多的学生社团，但是官方学生组织只有2个。大学可不是只有社团哦～

拾光文创和一般社团的区别: 首先，拾光文创的性质在严格意义上并不属于社团，是属于官方钦定的学生组织。在BNBU，社团一般由学生事务处（SAO）管理，拾光文创由IC管理。 其次，我们的资金来源会更充足（无需你转发朋友圈广告拉赞助亦或者为钱发愁）。 相反，我们还发钱！在拾光文创工作的工时都记录于BNBU官方的勤工俭学工时，并按照25Y/h的标准发放工资。 除此之外，在拾光文创你还可以拿到BNBU官方的工作证明（优秀工作员工）。这份工作证明不仅是学校社团经历的背书，也是你初步接触工作的见证。可所谓一石二鸟！

部门介绍

公关部 - IT组

工作内容:

  • 信息收集与整理，问卷制作
  • 大型活动提供设备技术支持
  • 程序系统设计与运维

能力要求:（以下二选一即可）

  • 熟练使用Excel、金山文档、问卷星等信息收集与整理软件
  • 有一定项目经验，会自己拉取上传github
  • 有部署服务器或者小程序经验（CDS、CST、DS、AI专业优先）

公关部 - HR组

工作内容:

  • 财务统计、薪酬调整
  • 人员调配、监管内部工作分配

能力要求:

  • ACCT、MHR、FIN、PRA专业优先
  • 优秀的沟通、表达和组织能力
  • 多任务平衡、团队意识强，善于察觉团队需求和潜力
  • 对图表数据敏感，擅长运用Excel进行高效数据处理

公关部 - 公关组

工作内容:

  • 内容审核与信息管理
  • 对外活动策划与执行协同
  • 招新策划与组织统筹
  • 社团舆情监测与反馈

能力要求:

  • 公共关系（PRA）、传媒（COMM）、市场营销（MKT）等专业优先
  • 具备良好的文字敏感度与内容判断能力
  • 具备活动策划与统筹能力
  • 沟通协调力强，善于跨组合作与对接外部资源
  • 思维清晰，具备基本数据处理能力

新媒体部 - 小红书组

工作内容:

  • 负责BNBU Souvenir官方小红书账号的图文及视频内容创作
  • 提升账号曝光度，协同团队进行账号运营

能力要求:

  • 小红书深度用户（专业不限）
  • 文笔流畅，具备创意思维
  • 擅长图片编辑与产品拍摄
  • 有小红书运营经验者优先（面试请自备作品/账号）

新媒体部 - 摄影组

工作内容:

主要负责围绕纪念品和活动进行视频拍摄和后期剪辑

能力要求:

  • 有拍摄视频经验以及摄影器材
  • 熟悉剪辑视频软件的使用

【投递要求】 请提前准备个人拍摄/剪辑的视频作品。

新媒体部 - 海报组

工作内容:

  • 定期设计节日主题海报
  • 配合其他部门活动制作宣传海报
  • 完成临时性创意平面设计任务

能力要求:

  • MAD、AIM等专业优先
  • 熟练使用至少一种设计工具（PS/AI优先）
  • 具备排版、配色、主题表达能力
  • 对视觉设计有敏锐度

新媒体部 - 公众号组

工作内容:

  • 负责推文的文案撰写、图文排版
  • 运营UIC Souvenir公众号

能力要求:

  • 文案/排版方面有所擅长
  • 熟练使用秀米、PS为加分项
  • 执行力强、不拖延
  • 为自己的任务负责任

商业部 - 纪念品组

工作内容:

  • 负责店内纪念品补货以及陈列
  • 帮助老师进行纪念品申领

能力要求:

  • 有较强的责任心，能够完成好任务
  • 能够较为熟练掌握word/excel等工作软件
  • 吃苦耐劳，能够工作一学年者优先

【年级要求】 只招大一大二同学

商业部 - 餐饮组

工作内容:

  • 负责协助餐饮工作人员完成点单、制作饮品、收餐盘等工作
  • 负责完成纪念品销售后取货、打包等工作
  • 工作时间为工作日用餐时段，每人每周至少需工作2小时

能力要求:

  • 热爱餐饮服务行业，有咖啡、饮品等制作基础者优先
  • 有良好的沟通社交能力和团队意识
  • 工作认真细致，不留美甲，能够合理安排时间
  • 可接受用餐高峰时段高强度工作

产品研发部 - 产品设计组

工作内容:

负责产出原创纪念品设计样图。

能力要求:

  • 有较强的设计能力
  • 能较为熟练使用AI/PS等设计软件
  • 热爱设计坚守原创不抄袭的原则底线

产品研发部 - 产品策划组

工作内容:

  • 参与文创产品的创意构思与设计提案
  • 能开发具有文化价值与市场潜力的产品
  • 跟进产品从设计到量产的全过程
  • 参与文创IP内容策划，增强产品附加值

能力要求:

  • 熟悉常见文创不同工艺，生产流程，有产品落地经验者优先
  • 具备敏锐的创意洞察力与良好的美学基础
  • 能提出新颖且可实现的产品创意
  • 具备一定的项目统筹能力，能协调多方资源

策划部 - 活动组

工作内容:

  • 负责策划方案的落地，需要紧密联系其他部门成员
  • 涉及到物资采买、活动场地布置、活动轮值等工作

能力要求:

  • 有较强的责任心，能够完成好任务
  • 沟通能力强且性格外向，能够和组员很好的合作
  • 面试通过后有试用期

策划部 - 策展组

工作内容:

负责每月橱窗、店内展位等地的策划与布置

能力要求:

  • 有较好的审美、规划和动手能力
  • CCM、MAD、CTV等专业优先
  • 面试通过后有试用期
